## Levantar el proyecto

Clonar el proyecto

```bash
  git clone https://github.com/JABEHAU/DownloaderProject
```

Ir al directorio raiz del proyecto

```bash
  cd DownloaderProject
```

Lanzar el proyecto

```bash
  python manage.py runserver
```
